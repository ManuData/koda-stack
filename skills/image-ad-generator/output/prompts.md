    Imagine that I want to do the same as I'm doing in the image-ad-generator but for videos in kling. What would be your approach         
                                                                                                                                  
⏺ Great question. Here's exactly how I'd approach it:                                                                                    
                                                                                                                                         
  ---                                                                                                                                    
  The Architecture
                                                                                                                                         
  Mirror image-ad-generator exactly — same folder structure, same CLI pattern, same art direction file compatibility. The only
  fundamental difference is Kling is async — you submit a task, poll for completion, then download. Imagen responds in seconds. Kling    
  takes 2–5 minutes per clip.
                                                                                                                                         
  ---             
  Folder Structure

  skills/
    video-generator/
      scripts/                                                                                                                           
        generate.py       ← same interface as image-ad-generator
      output/             ← .mp4 files land here                                                                                         
      README.md                                                                                                                          
  
  ---                                                                                                                                    
  The Script Logic

  1. PARSE INPUT
     Same 3 modes as image-ad-generator:                                                                                                 
     --visual PATH        → reads Generation Prompt from art direction .md                                                               
     --image PATH         → optional first frame (image-to-video mode)                                                                   
     "plain text"         → text-to-video mode                                                                                           
     '{"prompt": ...}'    → full JSON control
                                                                                                                                         
  2. AUTH         
     Kling uses JWT (not a plain API key)                                                                                                
     Generate token from KLING_ACCESS_KEY + KLING_SECRET_KEY                                                                             
     Sign with HMAC-SHA256, expires every 30min
                                                                                                                                         
  3. SUBMIT TASK  
     POST /v1/videos/text2video   ← no image                                                                                             
     POST /v1/videos/image2video  ← with --image flag                                                                                    
     Params: prompt, negative_prompt, duration (5|10),                                                                                   
             aspect_ratio, mode (std|pro), cfg_scale                                                                                     
                                                                                                                                         
  4. POLL FOR COMPLETION                                                                                                                 
     GET /v1/videos/{task_id}
     Poll every 15s until status = "succeed"                                                                                             
     Print progress dots so user sees it working
                                                                                                                                         
  5. DOWNLOAD + SAVE
     Download .mp4 from the result URL                                                                                                   
     Save to output/video_candidate_1.mp4 etc.                                                                                           
     Print JSON result same as image-ad-generator                                                                                        
                                                                                                                                         
  ---                                                                                                                                    
  New Parameters vs Image Generator                                                                                                      
                  
  ┌───────────────┬──────────┬────────────────────────────────┐
  │     Flag      │ Default  │             Notes              │
  ├───────────────┼──────────┼────────────────────────────────┤                                                                          
  │ --visual PATH │ —        │ Same as image-ad-generator     │
  ├───────────────┼──────────┼────────────────────────────────┤                                                                          
  │ --image PATH  │ —        │ First frame for image-to-video │
  ├───────────────┼──────────┼────────────────────────────────┤
  │ --duration    │ 5        │ 5 or 10 seconds                │
  ├───────────────┼──────────┼────────────────────────────────┤                                                                          
  │ --mode        │ standard │ standard or pro                │
  ├───────────────┼──────────┼────────────────────────────────┤                                                                          
  │ --num         │ 2        │ Max 2 concurrent tasks         │
  ├───────────────┼──────────┼────────────────────────────────┤                                                                          
  │ --aspect      │ 9:16     │ Same as image-ad-generator     │
  └───────────────┴──────────┴────────────────────────────────┘
